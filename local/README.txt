Στο παρόν directory υπάρχουν files που αφορούν μόνο στο τοπική υλοποίηση του
"sinergio" και ως εκ τούτου δεν είναι κοινοποιήσιμα. Οδηγούς για πολλά από τα
συγκεκριμένα files μπορούμε να βρούμε στο "lacol" directory. Χαρακτηριστικά
παραδείγματα τέτοιου είδους αρχείων είναι διάφορα configuration files, αρχεία
δεδομένων, test programs κλπ.
